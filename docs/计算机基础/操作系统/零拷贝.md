零拷贝 -- Zero Copy是一种计算机数据传输技术，减少数据在内存中的复制次数，从而提高系统性能。

在传统的数据传输过程中，数据通常需要经过多次复制：

1. 从磁盘读取数据到内核缓冲区
2. 从内核缓冲区复制到应用程序缓冲区
3. 从应用程序缓冲区复制回内核缓冲区
4. 从内核缓冲区写入到目标设备

而零拷贝技术避免这些冗余的数据复制操作，直接在内核空间完成数据传输，减少CPU使用和上下文切换，提高系统吞吐量和性能。

零拷贝的实现方式主要有：

+ mmap + write
+ sendfile
+ splice
+ 直接I/O
+ 写时复制技术

这种技术在处理大量数据传输时特别有效，例如网络服务器、文件服务器和数据库系统等应用场景。
