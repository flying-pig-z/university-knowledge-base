49. 字母异位词分组

```java
HashMap<String, List<String>> result = new HashMap<>();
```

+ String类型排序 

```java
String str = strs[i];
```


5. 最长回文子串

> dp数组：dp[首指针][尾指针] = 是否是回文子串
>
> 初始化：单个字母肯定是回文
>
> 递推公式：如果是两个，看s[i] == s[j]  
长度大于等于三 -- dp[i][j]是回文，则dp[i+1][j-1]是回文 && s[i] == s[j]
>
> 遍历方向：遍历方向，我是从右上角开始遍历
>
> 细节：在过程中记录起始点和字符串长度，便于最后截取 
>

1143. 最长公共子序列

> + dp数组定义：dp[i][j] 表示text1的前i个字符和text2的前j个字符的最长公共子序列长度  
> + 初始化：
>     -  dp[i][0] = 0: text2为空字符串时，最长公共子序列长度为0  
>     -  dp[0][j] = 0: text1为空字符串时，最长公共子序列长度为0  
>     -  因为Java数组默认值就是0，所以不需要显式初始化  
> + 遍历方向与递推公式：   
从左到右，从上到下，i和j的范围分别是[1,text1.length]和[1,test2.length]  
当前字符相同时，dp[i][j] = dp[i - 1][j - 1] + 1；当前字符不同时，取左边或上边的最大值。
