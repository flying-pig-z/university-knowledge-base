#### 11.盛最多水的容器
水容量 = 两条线高度的最小值 * 两条线的距离。暴力复杂度n的平方，但是我们可以使用双指针优化到n。

左指针从0，右指针从最右侧开始向中间移动，每次移动指向高度较小的指针（因为这个是短板，只有移动这个才有可能出现更优解），每次移动的时候计算容量，维护一个最大容量。

```java
class Solution {
    public int maxArea(int[] height) {
        // 两条线的最小值 * 两条线距离
        int max = 0;
        int left = 0;
        int right = height.length - 1;

        while(left<right){
            int waterContain = Math.min(height[left], height[right]) * (right - left);
            if(waterContain > max){
                max = waterContain;
            }

            if(height[left] < height[right]){
                left++;
            }else{
                right--;
            }
        }
        return max;
    }
}
```

#### 15.三数之和
对数组进行从小到大排序，便于后序的去重和双指针。

遍历数组，i为第一个数，然后使用双指针，left在i的右侧，right在数组末尾。

如果发现三数之和比0大就left指针就往右移动，如果发现比0小，right指针向左移动，发现等于0，就把这三个数加入结果集合中。

然后题目要求不重复，那我们这里去重逻辑就是先走一步，然后发现和之前重复 -- nums[i] == nums[i-1]，就走到不重复为止【走到尽头】。然后还要规定i left right的范围，防止越界。

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();

        Arrays.sort(nums);
        for(int i = 0;i<nums.length;i++){
            // 去掉重复的元素
            if(i>0 && nums[i] == nums[i-1]){
                continue;
            }
            int left = i+1;
            int right = nums.length - 1;
            while(left<right){
                int sum = nums[i] + nums[left] + nums[right];
                if(sum == 0){
                    result.add(new ArrayList<>(List.of(nums[i],nums[left],nums[right])));
                    left++;
                    // 去重 -- 如果是重复的，就一直走到不重复为止
                    while(left > 0 && left < nums.length && nums[left] == nums[left - 1]){
                        left++;
                    }
                }else if(sum > 0){
                    right--;
                    // 去重 -- 如果是重复的，就一直走到不重复为止
                    while(right != nums.length - 1 && right > 0 && nums[right] == nums[right + 1]){
                        right--;
                    }
                }else if(sum < 0){
                    left++;
                    // 去重 -- 如果是重复的，就一直走到不重复为止
                    while(left > 0 && left < nums.length && nums[left] == nums[left - 1]){
                        left++;
                    }
                }
            }

        }
        return result;
    }
}
```

其实去重i和得到结果后去重left就够了，因为前面两个不一样，最后就可以保证结果不同。所以可以去掉sum>0和sum<0中的去重，简化代码

```java
while(left<right){
    int sum = nums[i] + nums[left] + nums[right];
    if(sum == 0){
        result.add(new ArrayList<>(List.of(nums[i],nums[left],nums[right])));
        left++;
        // 去重 -- 如果是重复的，就一直走到不重复为止
        while(left > 0 && left < nums.length && nums[left] == nums[left - 1]){
            left++;
        }
    }else if(sum > 0){
        right--;
    }else if(sum < 0){
        left++;
    }
}
```

#### 42.接雨水
每一格能接的水 = min(左边最高, 右边最高) - 这格的height【要注意每格水都要大于0】，我们遍历每一格相加就可以了。

然后问题就转化成求左边最高和右边最高，如果在遍历的时候实时求解复杂度太高，我们提前得到每一格左边最高和右边最高的数组，维护一个最大量初始化为0从前往后遍历求左边最高，然后从后往前遍历就右边最高即可。

```java
    class Solution {
        public int trap(int[] height) {
            // 每一格能接的数量 = min(左边最高, 右边最高) - 这格的height
            
            // 定义左边最高和右边最高数组
            int[] leftMax = new int[height.length];
            int[] rightMax = new int[height.length];

            // 左边最高 
            int max = 0;
            for(int i = 0;i<height.length;i++){
                leftMax[i] = max;
                if(max < height[i])
                    max = height[i];
            }
            // 右边最高
            max = 0;
            for(int i = height.length-1;i>=0;i--){
                rightMax[i] = max;
                if(max < height[i])
                    max = height[i];
            }
            int sum = 0;
            for(int i = 0;i<height.length-1;i++){
                int current = Math.min(leftMax[i],rightMax[i]) - height[i];
                if(current>0)
                    sum = current + sum;
            }
            return sum;
        }
    }

```

## 滑动窗口
#### 3.无重复字符的最长子串
+ 使用左右指针(left和right)来控制窗口的大小，使用HashSet存储窗口元素窗口[便于查找有无重复字符]
+ 当遇到新的不重复字符时，扩大窗口(右指针右移)，当遇到重复字符时，收缩窗口(左指针右移)。
+ 维护最大长度，不断更新

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        if(s.length() == 0) return 0;
        int left = 0;
        int right = 0;
        int maxLength = 0;

        HashSet<Character> set = new HashSet<>();
        while(right < s.length()){
            // 如果集合没有right指针右移，更新最大长度,将当前元素装进集合里面
            if(!set.contains(s.charAt(right))){
                if((right - left + 1) > maxLength){
                    maxLength = right - left + 1;
                }
                set.add(s.charAt(right));
                right++;
            }
            // 如果集合中有，从集合中删除，左指针左移
            else{
                set.remove(s.charAt(left));
                left++;
            }
        }
        return maxLength;
    }
}
```

#### 438.找到字符串的所有字母异位词
判断是否是字母异位词是通过两个数组(数组有26格，统计26个字母的个数)比较是否相同，相同就是字母异位词。

暴力方法就是p先统计到一个数组。直接遍历，然后每一次将i和后面k个元素放到统计数组里面，然后比较。但是这样就会O(n*m)。

滑动窗口优化就是不重新装入数组，而是现在遍历到的新字符装进去，移除滑动窗口之外的旧字符。然后遍历+比较，优化到O(n)。

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        if(s.length()<p.length())
            return new ArrayList<>();

        List<Integer> result = new ArrayList<>();

        int[] pCount = new int[26];
        for(int i = 0;i<p.length();i++){
            pCount[p.charAt(i) - 'a']++;
        }
        // 初始化滑动窗口
        int[] windows = new int[26];
        for(int i = 0;i<p.length();i++){
            windows[s.charAt(i) - 'a']++;
        }
        if(Arrays.equals(windows,pCount)){
            result.add(0);
        }

        for(int i = p.length();i<s.length();i++){
            // 加入新字符
            windows[s.charAt(i) - 'a']++;
            // 移除旧字符
            windows[s.charAt(i-p.length()) - 'a']--;
            if(Arrays.equals(windows,pCount)){
                result.add(i-p.length()+1);
            }
        }
        return result;
    }
}
```

```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        HashMap<Integer, Integer> map = new HashMap<>();
        int[] prefix = new int[nums.length+1];
        prefix[0] = 0;
        map.put(0,1);

        int result = 0;
        for(int i = 0;i<nums.length;i++){
            prefix[i+1] = prefix[i] + nums[i];
            if(map.containsKey(prefix[i+1]-k)) result = result + map.get(prefix[i+1]-k);
            if(map.containsKey(prefix[i+1]))   map.put(prefix[i+1],map.get(prefix[i+1])+1);
            else map.put(prefix[i+1],1);
        }
        return result;
    }
}
```

## 普通数组
#### 53.最大子数组和
> 前缀和prefix(0) = 0，prefix(1) = prefix(0) + nums[0]
>

求出前缀和，然后子数组和 = prefix(j+1) - prefix(i)【从i到j的和】。

要求最大子数组和，就求每个prefix前面最小的prefix，这时候我们就从前往后遍历prefix数组，然后遍历的过程中维护一个变量minPrefixBefore记录当前prefix之前最小prefix，然后每一格的prefix - minPrefixBefore，其中的最小值就是最终的答案。

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int[] preSum = new int[nums.length+1];
        preSum[0] = 0;
        // 计算preSum的最小值
        for (int i = 1; i <= nums.length; i++) {
            preSum[i] = nums[i - 1] + preSum[i - 1];
        }
        // 要求最大值，那就减去最小值
        int minPreSum = 0;
        int maxResult = -1000000;
        for(int i = 1;i<=nums.length;i++){
            if(maxResult<(preSum[i] -minPreSum)) maxResult = preSum[i] - minPreSum;
            if(minPreSum>preSum[i]){
                minPreSum = preSum[i];
            } 
        }
        return maxResult;
    }
}
```

#### 56.合并区间
先对数组进行排序，然后合并的区间一定是相邻的。

我们用一个merged数组存放最终答案，遍历原数组。

第一个区间直接添加；接下去的区间，如果不能与merged数组最后一个元素合并直接添加；如果可以与merged数组合最后一个元素并且合并后的右边界大于原来右边界，则修改最后一个元素的右边界就可以了。

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        List<int[]> mergedList = new ArrayList<>();
        Arrays.sort(intervals, (a,b)->{
            return a[0] - b[0];
        });

        for(int i = 0;i<intervals.length;i++){
            if(i == 0){
                mergedList.add(intervals[i]);
                continue;
            }
            // 获取前一个区间
            int[] lastMegrge = mergedList.get(mergedList.size()-1);
            if(intervals[i][0]>lastMegrge[1]){
                // 如果不可以合并，直接添加
                mergedList.add(intervals[i]);
            }else{
                // 如果可以合并
                lastMegrge[1] = Math.max(intervals[i][1],lastMegrge[1]);
            }
        }
        int[][] realResult = new int[mergedList.size()][2];
        for(int i = 0;i<mergedList.size();i++){
            realResult[i][0] = mergedList.get(i)[0];
            realResult[i][1] = mergedList.get(i)[1];
        }
        return realResult;
    }
}
```

#### 238.除自身以外数组的乘积
计算乘积前缀和后缀，每一格的结果等于对应的前缀*后缀。

```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int[] prefix = new int[nums.length + 1];
        int[] postfix = new int[nums.length];

        prefix[0] = 1;
        for (int i = 1; i < nums.length; i++) {
            prefix[i] = prefix[i - 1] * nums[i - 1];
        }
        postfix[nums.length - 1] = 1;
        for (int i = nums.length - 2; i >= 0; i--) {
            postfix[i] = postfix[i + 1] * nums[i + 1];
        }
        
        int[] result = new int[nums.length];
        for (int i = 0; i < nums.length; i++) {
            result[i] = prefix[i] * postfix[i];
        }
        return result;
    }
}
```

## 矩阵
#### 73.矩阵置零
我这里先遍历哪些行和列有零，然后使用集合来保存需要置零的行和列，然后进行置零，空间复杂度为O(m+n)。

这里题解说还可以继续优化，使用第一行和第一列来报错需要置零的行和列。当然也要防止第一行和第一列本身为0，所以要用两个变量来保存第一行和第一列是否为零，如果为零，最后要将第一行和第一列置零得到正确答案。

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        Set<Integer> zeroRows = new HashSet<>();
        Set<Integer> zeroColumns = new HashSet<>();

        for(int i = 0;i<matrix.length;i++){
            for(int j = 0;j<matrix[i].length;j++){
                if(matrix[i][j] == 0){
                    zeroRows.add(i);
                    zeroColumns.add(j);
                }
            }
        }

        for(Integer row : zeroRows){
            for(int j = 0;j<matrix[row].length;j++){
                matrix[row][j] = 0;
            }
        }
        for(Integer column: zeroColumns){
            for(int i = 0;i<matrix.length;i++){
                matrix[i][column] = 0;
            }
        }
    }
}
```

#### 54.螺旋矩阵
使用方向变量 + 边界值 解决螺旋矩阵。

使用方向变量控制前进的方向，初始化为0（即右边），通过坐标x,y表示所在的位置。

每次循环沿着方向走，走之前检查一下有没有到达边界，到达边界则改变方向，缩小边界。（要在走之前检查，因为有时候刚开始就在边界）。

每次循环添加元素到答案集合，当答案集合等于数组长度时候退出。

```java
class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        List<Integer> result = new ArrayList<>();
        
        int up = 0, down = matrix.length - 1, left = 0,right = matrix[0].length - 1; // 边界
        int direction = 0; // 0-右边，1-下，2-左，3-上
        int x = 0,y = 0; // 坐标
        while(true){
            result.add(matrix[x][y]);
            // 集合元素达到一定数量退出
            if(result.size() == matrix.length*matrix[0].length)
                break;
            // 到达边界 -- 改变方向，缩小边界
            if(direction == 0 && y == right){
                direction=1;
                up++;
            }else if(direction == 1 && x == down){
                direction = 2;
                right--;
            }else if(direction == 2 && y == left){
                direction=3;
                down--;
            }else if(direction == 3 && x==up){
                direction=0;
                left++;
            }
            // 沿着方向走
            if(direction == 0) y++;
            else if(direction == 1) x++;
            else if(direction == 2) y--;
            else if(direction == 3) x--;
        }
        return result;
    }
}
```

#### 48.旋转图像
第一步：沿着从左上到右下的主对角线翻转，遍历矩阵的上半部分(包括对角线)，交换 matrix[i][j] 和 matrix[j][i] 的值。

第二步：沿中垂线左右翻转，遍历矩阵每一行的左半部分，交换 matrix[i][j] 和 matrix[i][n-1-j] 的值。

```java
class Solution {
    public void rotate(int[][] matrix) {
        // 沿着从左上到右下的主对角线翻转
        for (int i = 0; i < matrix.length; i++) {
            for (int j = i; j < matrix[i].length; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
        // 沿着中垂线左右翻转
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < (matrix.length + 1) / 2; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[i][matrix.length - 1 - j];
                matrix[i][matrix.length - 1 - j] = temp;
            }
        }
    }
}
```

#### 240.搜索二维矩阵 II
最简洁的写法是对每一行都进行二分查找，复杂度是O(mlongn)。

> 这里要清楚Java怎么对数组进行二分查找，如果没找到返回什么
>

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        for (int i = 0; i < matrix.length; i++) {
            if (Arrays.binarySearch(matrix[i], target) >= 0)
                return true;
        }
        return false;
    }
}
```

更优的解法是利用这个矩阵的特性，从右上角开始搜索，如果比这个目标小就往下移动，如果比目标大就往左移动。

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        // 从右上角开始搜索，如果比这个目标小就往下移动，如果比目标大就往左移动
        int x = matrix.length - 1;
        int y = 0;
        while (x >= 0 && y <= matrix[0].length - 1) {
            if (target == matrix[x][y])
                return true;
            else if (target > matrix[x][y])
                y++;
            else if (target < matrix[x][y])
                x--;
        }
        return false;
    }
}
```

#### 206.反转链表
先遍历得到链表数组，然后从后往前遍历，将每一个节点指向前一个节点，就得到了反转链表。

```java
class Solution {
    public ListNode reverseList(ListNode head) {
        if (head == null)
            return null;
        List<ListNode> listNodes = new ArrayList<>();
        ListNode now = head;
        while (now != null) {
            listNodes.add(now);
            now = now.next;
        }
        for (int i = listNodes.size() - 1; i > 0; i--) {
            listNodes.get(i).next = listNodes.get(i-1);
```

#### 33.搜索旋转排序数组
与普通二分查找要复杂一点：

1. 判断mid在哪个排序子数组中
2. 如果左半部分有序，就根据有序的特点判断目标值是否在左半部分。  
在左半部分，缩小搜索范围到左半部分；不在左半部分，搜索右半部分。
3. 如果右半部分有序，就根据有序的特点判断目标值是否在右半部分。  
在右半部分，缩小搜索范围到右半部分；不在右半部分，搜索右半部分。

> 如何判断左半部分有序 --  nums[left] <= nums[mid]  
>
>  判断目标值是否在左半部分   -- nums[left]<= target && target < nums[mid]
>
>  判断目标值是否在右半部分  -- nums[mid]< target && target <= nums[right]
>

```java
class Solution {
    public int search(int[] nums, int target) {
        int left = 0;
        int right = nums.length - 1;
        while (left <= right) {
            int mid = (left + right) / 2;
            // 如果mid是目标值，直接返回
            if (nums[mid] == target) return mid;
            // 判断mid在哪个有序数组中
            if (nums[left] <= nums[mid]) {
                // 判断target是否在左半部分
                if (target >= nums[left] && target <= nums[mid]) {
                    right = mid - 1;
                } else {
                    left = mid + 1;
                }
            } else {
                // 判断target是否在右半部分
                if (target <= nums[right] && target >= nums[mid]) {
                    left = mid + 1;
                } else {
                    right = mid - 1;
                }
            }
        }
        return -1;
    }
}
```

#### 155.最小栈
只要维护一个变量记录最小值。在插入删除元素时候更新最小值。

插入时候直接把插入的元素与原来最小值比较进行更新。

删除的时候移除元素，使用辅助栈来遍历栈上元素获得最小值。

```java
class MinStack {
    
    int min = Integer.MAX_VALUE;
    Stack<Integer> stack = new Stack<>();
    
    public MinStack() {
    }
    
    public void push(int val) {
        stack.push(val);
        if(val<min) min = val;
    }
    
    public void pop() {
        int deleteVal = stack.pop();
        if(deleteVal == min){
            Stack<Integer> temp = new Stack<>();
            min = Integer.MAX_VALUE;
            while(!stack.isEmpty()){
                int val = stack.pop();
                if(val<min) min = val;
                temp.add(val);
            }
            while(!temp.isEmpty()){
                stack.add(temp.pop());
            }
        } 
    }
    
    public int top() {
        return stack.peek();
    }
    
    public int getMin() {
        return min;
    }
}
```

#### 394.字符串解码（只有代码）
看代码抄几遍估计就记住了

```plain
class Solution {
    public String decodeString(String s) {
        // 用于存储重复次数的栈
        Stack<Integer> countStack = new Stack<>();
        // 用于存储之前已经处理过的字符串的栈
        Stack<String> stringStack = new Stack<>();
        // 当前正在处理的字符串
        String currentString = "";
        // 当前正在解析的数字(重复次数)
        int k = 0;
        
        // 逐个字符处理输入字符串
        for (char ch : s.toCharArray()) {
            if (Character.isDigit(ch)) {
                // 如果是数字，计算完整的重复次数
                // 比如 "23" 会先读入 '2'，k = 2，再读入 '3'，k = 2*10 + 3 = 23
                k = k * 10 + (ch - '0');
            } else if (ch == '[') {
                // 遇到左括号，说明即将进入一个新的重复子串
                // 1. 将当前累计的重复次数k压入countStack
                countStack.push(k);
                // 2. 将当前已构建的字符串压入stringStack，下一步会构建新的子串
                stringStack.push(currentString);
                // 3. 重置k和currentString，准备解析新的重复子串
                k = 0;
                currentString = "";
            } else if (ch == ']') {
                // 遇到右括号，说明一个重复子串解析完毕，需要进行重复拼接操作
                // 1. 从stringStack弹出之前的字符串
                String previousString = stringStack.pop();
                // 2. 从countStack弹出重复次数
                int repeatTimes = countStack.pop();
                
                // 3. 将当前子串重复指定次数
                String repeated = "";
                for (int i = 0; i < repeatTimes; i++) {
                    repeated += currentString;
                }
                
                // 4. 将重复后的子串与之前的字符串拼接，形成新的当前字符串
                currentString = previousString + repeated;
            } else {
                // 如果是普通字母，直接添加到当前字符串
                currentString += ch;
            }
        }
        
        // 返回最终解码后的字符串
        return currentString;
    }
}
```

#### 45.跳跃游戏（二）
我们用一种"看得最远跳得最远"的策略：

1. **当前可跳区域**：你现在站的位置，加上你能跳的最大步数，组成了你"当前可以选择落脚"的区域
2. **选择下一跳**：在这个可选区域内的每个位置，都计算一下"如果跳到这里，下一步最远能到哪"
3. **贪心选择**：选择那个"下一步能跳得最远"的位置作为你的下一跳

具体步骤：

+ 从位置0开始遍历
+ 设定当前这一跳的"覆盖范围"，初始尾0（end变量）
+ 在这个覆盖范围内都可以去尝试，不断更新"下一跳最远能到哪"（maxPosition变量）
+ 当我们走到当前这一跳的范围时，就必须进行跳跃了，选择下一跳能到达的最远距离进行跳跃 -- 更新覆盖范围/边界为这个最远距离，并跳跃次数+1，如果发现超出数组范围说明已经到达直接范围。

> 要记得数组长度为1直接返回0，不用跳跃。
>

```java
class Solution {
    public int jump(int[] nums) {
        // 基本思想：站的更远看的更远

        // 已经在终点，不需要跳跃
        if (nums.length == 1)
            return 0;

        int step = 0; // 步数
        int maxPos = 0; // 最远到达距离
        int end = 0; // 这一跳的覆盖范围
        // 从0开始遍历
        for (int i = 0; i <= nums.length - 1; i++) {
            // 更新在这一跳范围能下一跳能跳到的最大距离
            if (i + nums[i] > maxPos)
                maxPos = i + nums[i];
            // 到达这一跳的覆盖范围，就要做出选择，开始跳跃
            if (i == end) {
                // 选择能跳到最远的地方
                end = maxPos; // 更新边界
                step++; // 增加步数
                if (maxPos >= nums.length - 1) break; // 如果这一步能跳到终点之后直接返回
            }
        }
        return step;
    }
}
```

#### 763.划分字母区间
核心思想：每个区间里面的字符肯定都要是最后一次出现，然后区间要尽可能小，就是当找到一个区间里面的字符都是最后一次出现的时候，就不要进一步扩张了。

我们先遍历一遍数组，记录每个字符最后出现的位置。

维护start表示片段的开始，end表示片段的结束。

然后再次遍历数组，遍历的时候更新片段的结束end为当前字符串和原来end的最大值。

然后如果i==end，代表end里面的字符都是最后一次出现，这时候就可以把这个片段的长度添加到结果集中（不需要进一步扩展，保证区间尽可能多），并把start更新成下一个片段起点，寻找下一个片段。

```java
class Solution {
    public List<Integer> partitionLabels(String s) {
        // 每个区间里面的字符肯定都要是最后一次出现
        // 然后区间要尽可能小，就是当找到一个区间里面的字符都是最后一次出现的时候，就不要进一步扩张了

        // 记录每个字符最后出现的位置
        int[] lastPos = new int[26];
        for (int i = 0; i < s.length(); i++) {
            lastPos[s.charAt(i) - 'a'] = i;
        }

        List<Integer> result = new ArrayList<>(); // 结果集
        int start = 0; // 片段的开始位置
        int end = 0; // 片段的结束位置

        for (int i = 0; i < s.length(); i++) {
            // 当发现end内的字符不是最后出现，更新end为这个字符的最后出现位置
            end = Math.max(end, lastPos[s.charAt(i) - 'a']); 
            // 当i等于end，代表end里面的字符都是最后一次出现，这时候就可以把这个片段的长度添加到结果集中
            // 并把start更新成下一个片段起点，寻找下一个片段（不需要进一步扩展，保证区间尽可能多）
            if (i == end) {
                result.add(end - start + 1);
                start = end + 1;、
            }
        }

        return result;
    }
}
```

```java
        return result;
```

```java
    public void backtracking(int[] candidates, int target, int startIndex) {
        if (sum >= target) {
            if (sum == target) {
                result.add(new ArrayList<>(subResult));
            }
            return;
        }

        for (int i = startIndex; i < candidates.length; i++) {
            subResult.add(candidates[i]);
            sum = sum + candidates[i];
            backtracking(candidates, target, i);
            subResult.removeLast();
            sum = sum - candidates[i];
        }
    }
}
```

```java
            return;
```

#### 543.二叉树的直径
**核心思路：求所有节点中左右子树深度和的最大值**

**实现方法就是实现一个求深度的递归函数 -- 节点的深度 = max(左子树深度,右子树深度) + 1，并且维护一个变量作为最大直径，每一次都更新为 -- 左子树深度 + 右子树深度 。**

```java
class Solution {
    int diameter = 0;

    public int diameterOfBinaryTree(TreeNode root) {
        // 求所有节点中左右子树深度和的最大值
        maxDepth(root);
        return diameter;
    }
    private int maxDepth(TreeNode root){
        if(root == null)
            return 0;
        
        int leftMaxDepth = maxDepth(root.left);
        int rightMaxDepth = maxDepth(root.right);

        int totalMaxDepth = leftMaxDepth + rightMaxDepth;
        if(totalMaxDepth>diameter)
            diameter = totalMaxDepth;

        return Math.max(leftMaxDepth, rightMaxDepth) + 1;
    }
}
```

#### 105.从前序与中序遍历序列构造二叉树

2. 中序遍历的顺序是：左子树 -> 根节点 -> 右子树

```java
class Solution {

    HashMap<Integer, Integer> map = new HashMap<>();

    public TreeNode buildTree(int[] preorder, int[] inorder) {
        for(int i = 0;i<inorder.length;i++){
            map.put(inorder[i],i);
        }
        return helper(preorder,0,preorder.length-1,inorder,0,inorder.length-1);
    }

    private TreeNode helper(int[] preorder,int preBegin, int preEnd,int[] inorder, int inBegin, int inEnd ){
        if(preBegin>preEnd)
            return null;
        TreeNode root = new TreeNode(preorder[preBegin]);

        Integer inorderRoot = map.get(preorder[preBegin]);
        Integer leftNum = inorderRoot - inBegin;
        Integer rightNum = inEnd - inorderRoot;
        root.left = helper(preorder,preBegin+1,preBegin+leftNum,inorder,inBegin,inorderRoot-1);
        root.right = helper(preorder,preEnd-rightNum+1,preEnd,inorder,inorderRoot+1,inEnd);

        return root;
        
    }
}
```

#### 437.路径总和三
> 好难啊，第一次接触到双层递归
>

1. 外层递归：遍历二叉树的每个节点，把每个节点都当作起点
2. 内层递归：以当前节点为起点，计算向下所有可能的路径中和为targetSum的数量

```java
class Solution {
    public int pathSum(TreeNode root, int targetSum) {
        if(root == null){
            return 0;
        }
        // 计算以当前节点开始的路径数
        int ret = dfs(root,targetSum);
        // 递归计算左右结点
        return ret + pathSum(root.right, targetSum) + pathSum(root.right, targetSum);
    }
    // 计算以某个节点开始的路径和
    private int dfs(TreeNode node, long targetSum){
        if(node == null) return 0;
        int count = 0;
        if(node.val == targetSum) count++;
        count = count + dfs(node.left, targetSum - node.val) + dfs(node.right, targetSum - node.val); 
        return count;
    }
}
```

#### 236.二叉树的最近公共祖先
用哈希表存储每个结点的父节点，然后找出p的所有祖先与q的所有祖先，然后找到最近公共祖先。

这里还进行了优化，先找到p的所有祖先，放到哈希集合中，然后在回溯q的祖先中，发现也是p祖先的直接返回。

```java
class Solution {
    Map<TreeNode, TreeNode> map = new HashMap<>();

    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        // 用哈希表存储每个节点的父节点
        preorder(root);
        // 直接找出所有祖先，然后遍历查找公共祖先
        TreeNode current = p;
        HashSet<TreeNode> pFather = new HashSet<>();
        while (current != null) {
            pFather.add(current);
            if (map.containsKey(current))
                current = map.get(current);
            else
                break;
        }
        current = q;
        while (current != null) {
            if (pFather.contains(current))
                return current;
            else {
                if (map.containsKey(current))
                    current = map.get(current);
                else
                    break;
            }
        }
        return null;
    }

    public void preorder(TreeNode root) {
        if (root == null)
            return;
        map.put(root.left, root);
        map.put(root.right, root);
        preorder(root.left);
        preorder(root.right);
    }
}
```

#### 279.完全平方数
完全背包问题

核心：对于每个数 i，我们遍历所有小于等于 i 的完全平方数（j*j），尝试用这个完全平方数来更新 dp[i]。

```java
class Solution {
    public int numSquares(int n) {
        // 创建一个大小为 n+1 的 dp 数组
        // dp[i] 表示和为 i 的完全平方数的最少数量
        int[] dp = new int[n+1];
        // 初始化 dp 数组，假设最坏情况下每个数都由最小完全平方数1组成，所以 dp[i] = i它本身
        for(int i = 1;i<=n;i++){
            dp[i] = n;
        }
        // 对于每个数 i，我们遍历所有小于等于 i 的完全平方数（j*j）
        // 尝试用这个完全平方数来更新 dp[i]
        for(int i = 1;i<=n;i++){
            for(int j = 1;j*j<=i;j++){
                // 状态转移方程：dp[i] = min(dp[i], dp[i - j*j] + 1) 
                // dp[i] 表示当前的最小数量
                // dp[i - j*j] + 1表示使用 j*j 替换后剩余数字所需的最小数量加1
                dp[i] = Math.min(dp[i], dp[i-j*j] + 1);
            }
        }
        // dp[n]为所求结果
        return dp[n];
    }
}
```

#### 322.零钱兑换
完全背包问题

核心：遍历金额，再遍历硬币，看看选择该硬币能不能减少该金额所需硬币数。

注意硬币面额要小于等于当前要凑的金额才有尝试的意义，所以要加个条件判断。

```java
import java.util.Arrays;

public class CoinChange {
    public int coinChange(int[] coins, int amount) {
        // 创建dp数组，dp[i]表示凑成金额i所需的最少硬币个数
        int[] dp = new int[amount + 1];
        
        // 初始化dp数组，除了dp[0]=0外，其余设为一个不可能达到的大值
        // 选择amount+1是因为硬币最小面额为1，所以最多需要amount个硬币
        Arrays.fill(dp, amount + 1);
        
        // 基础情况：凑成金额0需要0个硬币
        dp[0] = 0;
        
        // 计算每个金额所需的最少硬币数
        for (int i = 1; i <= amount; i++) {
            // 尝试每种硬币
            for (int coin : coins) {
                // 硬币面额要小于等于当前要凑的金额才有尝试的意义
                if (coin <= i) {
                    // 状态转移方程：看看选择硬币能不能减少所需硬币数
                    dp[i] = Math.min(dp[i], dp[i - coin] + 1);
                }
            }
        }
        
        // 如果dp[amount]仍然是初始值，说明无法凑成总金额，返回-1
        return dp[amount] > amount ? -1 : dp[amount];
    }
}
```

```java
        for (int i = 1; i < n; i++) {
            for (int j = 0; j < i; j++) {
                if (nums[i] > nums[j]) {
                    // 如果nums[i]大于前面的某个元素nums[j]，则可以将nums[i]接在以nums[j]结尾的子序列后面
                    dp[i] = Math.max(dp[i], dp[j] + 1);
                }
            }
            // 更新全局最长长度
            maxLength = Math.max(maxLength, dp[i]);
        }
        
        return maxLength;
    }
}
```

#### 152.乘积最大子数组
这里与上面不同，要维护两个变量，因为要考虑负数的情况。

记忆口诀："当前值，乘最大，乘最小，三者取"

```java
class Solution {
    public int maxProduct(int[] nums) {
        // 初始化最大值为第一个元素
        int globalMax = nums[0];
        
        // 维护以上一个位置结尾的子数组的最大乘积和最小乘积
        int preMax = nums[0];
        int preMin = nums[0];
        
        // 从第二个元素开始遍历
        for (int i = 1; i < nums.length; i++) {
            // 保存当前的preMax，因为在计算preMin时会用到
            int temp = preMax;
            
            // 考虑到负数的情况，当前位置的最大乘积可能来自:
            // 1. 当前元素自己
            // 2. 当前元素 * 前一个位置的最大乘积
            // 3. 当前元素 * 前一个位置的最小乘积（如果当前元素为负数，则与负数相乘可能变为最大值）
            preMax = Math.max(nums[i], Math.max(preMax * nums[i], preMin * nums[i]));
            
            // 同理，当前位置的最小乘积
            preMin = Math.min(nums[i], Math.min(temp * nums[i], preMin * nums[i]));
            
            // 更新全局最大值
            globalMax = Math.max(globalMax, preMax);
        }
        
        return globalMax;
    }
}
```

#### 416.分割等和子集
题解：

[https://www.yuque.com/flyingpig-pbfgd/gig22l/fvik0zae285wi7wy](https://www.yuque.com/flyingpig-pbfgd/gig22l/fvik0zae285wi7wy)

```java
class Solution {
    public boolean canPartition(int[] nums) {
        // 问题转化：找出和为二分之一总和的子集
        // 计算数组总和
        int sum = 0;
        for (int num : nums) {
            sum = sum + num;
        }
        // 不能平分，说明有和为二分之一的子集
        if(sum % 2 != 0) return false;
        int target = sum / 2;
        boolean[] dp = new boolean[target + 1];
        dp[0] = true;
        // 数组中一个个数字试过去
        for (int num : nums) {
            // dp数组遍历 -- 选择不选择这个数字
            // 因为小于target的数字就考虑了，因为不可以选择这个数字
            // 反向遍历是为了防止同一个数字被使用多次
            for (int i = target; i >= num; i--) {
                dp[i] = dp[i] || dp[i - num];
            }
            // 如果这个dp[target] = true，说明子集已经找到，直接返回
            if (dp[target])
                return true;
        }
        // 遍历完说明子集没有找到
        return false;
    }
}
```

#### 1143.最长公共子序列
+ dp数组定义：dp[i][j] 表示text1的前i个字符和text2的前j个字符的最长公共子序列长度  
+ 初始化：
    - dp[i][0] = 0: text2为空字符串时，最长公共子序列长度为0  
    - dp[0][j] = 0: text1为空字符串时，最长公共子序列长度为0  
    - 因为Java数组默认值就是0，所以不需要显式初始化  
+ 遍历方向与递推公式：   
从左到右，从上到下，i和j的范围分别是[1,text1.length]和[1,test2.length]  
当前字符相同时，dp[i][j] = dp[i - 1][j - 1] + 1；当前字符不同时，取左边或上边的最大值。

```java
class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        int[][] dp = new int[text1.length()+1][text2.length()+1];

        for(int i = 1;i<=text1.length();i++){
            for(int j = 1;j<=text2.length();j++){
                if(text1.charAt(i-1) == text2.charAt(j-1)){
                    dp[i][j] = dp[i-1][j-1]+1;
                }else{
                    dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]);
                }
            }
        }
        return dp[text1.length()][text2.length()];
    }
}
```

```java
class Solution {
    public int minDistance(String word1, String word2) {
        // dp[i][j]表示word1的前i个字符转化成word2的前j个字符的最少操作数
        int[][] dp = new int[word1.length()+1][word2.length()+1];
        // 初始化dp[0][j]等于插入j次，最小操作数为j
        for(int j = 0;j<=word2.length();j++){
            dp[0][j] = j;
        }
        // 初始化dp[i][0]等于删除i次，最小操作数为i
        for(int i = 0;i<=word1.length();i++){
            dp[i][0] = i;
        }
        // dp[i][j] = min(替换操作，插入操作，删除操作)
        // 替换：dp[i-1][j-1]+1 插入：dp[i][j-1]+1 删除：dp[i-1][j]+1
        for(int i = 1;i<=word1.length();i++){
            for(int j = 1;j<=word2.length();j++){
                if(word1.charAt(i-1) == word2.charAt(j-1))
                    dp[i][j] = dp[i-1][j-1];
                else
                    dp[i][j] = Math.min(dp[i-1][j-1]+1,Math.min(dp[i][j-1]+1, dp[i-1][j]+1));
            }
        }
        return dp[word1.length()][word2.length()];

    }
}
```

#### 136.只出现一次的数字

```java
class Solution {
    public int singleNumber(int[] nums) {
        Map<Integer, Integer> map = new HashMap<>();
        for(int i = 0;i<nums.length;i++){
            int num = nums[i];
            if(map.containsKey(num)){
                map.put(num, map.get(num) + 1);
            }else{
                map.put(num, 1);
            }
        }
        for(int i =0;i<nums.length;i++){
            if(map.get(nums[i]) == 1){
                return nums[i];
            }
        }
        return nums[0];
    }
}
```

```java
class Solution {
    public void nextPermutation(int[] nums) {
        int i = nums.length - 2;
        // 从右向左寻找第一个递减的位置（即nums[i] < nums[i+1]）
        while (i >= 0 && nums[i] >= nums[i + 1]) {
            i--;
        }
        
        // 如果找到了这样的位置
        if (i >= 0) {
            int j = nums.length - 1;
            // 从右向左找到第一个大于nums[i]的元素
            while (j >= 0 && nums[j] <= nums[i]) {
                j--;
            }
            
            // 交换nums[i]和nums[j]
            int temp = nums[i];
            nums[i] = nums[j];
            nums[j] = temp;
        }
        
        // 将i之后的部分反转
        reverse(nums, i + 1);
    }
    
    private void reverse(int[] nums, int start) {
        int left = start;
        int right = nums.length - 1;
        while (left < right) {
            int temp = nums[left];
            nums[left] = nums[right];
            nums[right] = temp;
            left++;
            right--;
        }
    }
}
```

#### 287.寻找重复数
我的评价是：开背！

快慢指针，但是与常规快慢指针不同，这里是用跳的，而且跳的方式是跳到数组指向的位置。

初始化为nums[0]，然后slow往前跳一格，而fast往前跳两格，直到下次相遇。因为有重复数字，所以有环，所以最后都会相遇。

然后slow回到nums[0]。然后每个指针往前一步，最后相遇就是结果。

```java
public int findDuplicate(int[] nums) {
    // 初始化，都从同一个起点开始
    int slow = nums[0];
    int fast = nums[0];
    
    // 第一阶段：找到相遇点
    do {
        slow = nums[slow];         // 慢指针走一步
        fast = nums[nums[fast]];   // 快指针走两步
    } while (slow != fast);
    
    // 第二阶段：找到环的入口
    slow = nums[0];                // 慢指针回到起点
    while (slow != fast) {
        slow = nums[slow];         // 两个指针都走一步
        fast = nums[fast];
    }
    
    return slow;                   // 返回重复的数
}
```
