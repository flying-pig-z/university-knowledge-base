#### 151.反转字符串中的单词
注意分割字符串的API -- split，同时注意会有""字符串的情况。

把这个字符串使用split函数通过空格切割成一个个单词，去除""字符，再反向拼接

```java
class Solution {
    public String reverseWords(String s) {
        // 思路：把这个字符串使用split函数通过空格切割成一个个单词，再反向拼接
        String[] strings = s.split(" ");
        // 去除""字符
        ArrayList<String> stringArray = new ArrayList<>();
        for(int i = 0;i<strings.length;i++){
            if(strings[i] != "") stringArray.add(strings[i]);
        }
        // 反向拼接
        String result = "";
        for(int i = stringArray.size() -1;i>0;i--){
            result = result + stringArray.get(i) + " ";
        }
        result = result + stringArray.get(0);
        return result;
    }
}
```
