## 一. 计数排序算法
基本思想：统计每个输入元素的个数，然后根据这些计数值重构原数组。

使用范围：需要知道元素大小范围，就是[最大值](https://so.csdn.net/so/search?q=%E6%9C%80%E5%A4%A7%E5%80%BC&spm=1001.2101.3001.7020)是多少。

## 二. 基数排序
使用场景：只适用于整数

基数排序思路:  

1. 将待排序序列从个位到高位组织进不同的桶当中  
【注：这是最低位优先法 (LSD), 而最高位优先法(MSD) 是从高位到低位】  
2. 再按照从左到右的顺序从桶中将元素拿出  
3. 经过 n 轮操作后  
4. 序列整体有序
