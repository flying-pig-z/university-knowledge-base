> 之前学C语言写的笔记
>



一.只要有三种基本逻辑结构（顺序，选择，循环）就能实现各种复杂逻辑，再配上变量，操作符，指针，就能实现任何运算。

加上输入，输出函数，就可以实现基础的人机交互

再加上其他高级函数，如控制台交互函数就可以制作控制台程序，Windows函数就可以制作Windows客户端软件

> 有些函数底层是汇编语言要和硬件结合，高级语言必须要引用
>



二.数组

+ 便于数据的管理

> 以贪吃蛇游戏制作中打印地图为例，如果只有x,y和部分变量，那么程序将会异常麻烦，而如果将a[10][10]化为100个变量，那么也会异常麻烦。（虽然也不是不可以）
>
> 而数组是的数据变得有序，通过索引，与for循环结合，变得异常方便，同时也大幅简化了代码量。
>

+ 给数据以标签数字，从而便于对数据进行操作
+ 数组与链表都是数据结构，本质上都是指针，各有优缺点，便于对数据进行处理。



三.函数

+ 结构化变成，将小事封装成大事，使更有逻辑
+ 重复使用一段代码



四.指针

+ 动态分配内存，制作链表等数据结构
+ 函数中可以修改本地变量（你或许会说设置全局变量不就好了，但是那是不安全的）



五.类型和运算

+ 不同类型的本质是相同的，都是一串二进制数，只不过按照不同的空间大小和不同划分方式形成不同类型，但是本质上都是二进制数
+ 人类世界用的是四则运算，四则运算也可以实现所有复杂运算，而取余是为了方便，它截取了除后的后半部分，而除法是截取前半部分，浮点转整型类型转换也是一种截取，截取小数点前的部分
+ 整型转换与printf，scanf中转换不一样，遵循一定规则，其中浮点转为整数是截取，而不同精度浮点之间转换规则较为复杂
