## 运行时数据
### 线程不共享
**程序计数器：** 记录当前线程正在执行的字节码指令地址，用于线程切换后能恢复到正确的执行位置

**虚拟机栈：** 存储方法调用时的局部变量、操作数栈、方法出口等信息，每个方法调用对应一个栈帧

**本地方法栈：** 为虚拟机调用Native方法（C/C++方法）提供服务的栈空间

### 线程共享
**堆：** 存放所有对象实例和数组，是垃圾收集器管理的主要区域，所有线程共享

**方法区：** 存储类信息、常量、静态变量、即时编译器编译后的代码等数据，所有线程共享

### 直接内存
不属于JVM内存，是直接向操作系统申请的内存空间，常用于NIO操作以提高性能
